## Connect to KBVE IRC

Sign in with your KBVE account using Discord, GitHub, or Twitch to start chatting.

### Features

- **Real-time messaging** powered by Ergo IRC
- **OAuth authentication** via Discord, GitHub, and Twitch
- **WebSocket gateway** for browser-based IRC access
