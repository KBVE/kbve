## Effect

**Rune Storage:**
- Holds 3 types of runes
- 16,000 of each type
- Counts as one inventory slot

## Usage

**Magic:**
- Essential for PvM mages
- Saves inventory space
- Works with autocast

**Slayer:**
- High alch + combat spells
- Bursting/barraging tasks

**PvP:**
- Multiple spell types in one slot
- Tribrid setups

## Upgrades

Can be upgraded to Divine rune pouch with:
- 75 Crafting
- Needle
- Thread of Elidinis

## Market Strategy

**Notes:**
- Untradeable directly
- Buy note from GE
- Essential magic utility
