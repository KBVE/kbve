## Usage

The Hallowed token is a consumable item used within the Hallowed Sepulchre minigame. When activated, it extends the player's timer by 1 minute, allowing more time to complete floors and loot coffins.

### Strategy

- Best used on Floor 4-5 when time is running low
- Can be activated at any time during a run
- Multiple tokens can be used per run
- Useful for learning the minigame or attempting speedruns

## About

Hallowed tokens provide extra time flexibility in the Hallowed Sepulchre, particularly valuable when:
- Learning new floors
- Attempting to loot all coffins
- Going for pet chance by completing full runs
