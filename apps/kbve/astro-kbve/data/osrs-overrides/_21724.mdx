## Obtaining

Dropped by Gargoyles during Slayer tasks.

**Drop rate:** 1/150

**Requirements:**
- 75 Slayer to kill Gargoyles
- Active Slayer task

## Usage

**Unlocks Grotesque Guardians:**
- Use on Grotesque Guardian door (Slayer Tower rooftop)
- Permanent unlock - only needed once
- Subsequent drops are automatically deleted

**One-time use:**
- After unlocking, key is consumed
- Cannot be banked
- "You don't need that" message if picked up after unlocking
