## Market Flip Strategy

**Buy Unstrung, Sell Strung:**
1. Buy Yew longbow (u) on GE
2. Buy Bow string on GE
3. String into Yew longbow
4. Sell or High Alch

**Profit Calculation:**
- `Yew longbow price - (Yew longbow (u) + Bow string) = Stringing profit`
- High Alch value: 768 GP

**Alch Flipping:**
- If `Yew longbow (u) + Bow string + Nature rune < 768 GP` = profit from alching
