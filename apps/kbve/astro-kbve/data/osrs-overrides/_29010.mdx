## Obtaining

The Eclipse moon helm is obtained from the Lunar Chest in Neypotzli with a 1/224 drop rate.

**Requirements:** 75 Ranged, 50 Defence to equip

## Set Effect

When wearing the full Eclipse moon armour set with the Eclipse atlatl:
- Successful attacks have a 20% chance to inflict burn damage on non-immune targets
