## Examine

> *"Enchanted Wizards robes."*

## Obtaining

- **Treasure Trails** — hard clue reward (1/1,625 from reward casket)
- **Grand Exchange** — tradeable
- **Storage** — can be stored in costume room treasure chest

The enchanted robe is exclusively a hard Treasure Trails reward. It shares identical stats with the mystic robe bottom but has a unique visual appearance.

## Usage

- Cosmetic alternative to the mystic robe bottom
- **Treasure Trails** fashionscape outfit
- Can be stored in POH costume room
- Same combat utility as mystic robe bottom in every scenario
