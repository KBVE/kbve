## Set Effect

When the full Hallowed graceful set is worn:
- Run energy restoration rate increased by 30%
- Total weight reduction: -25 kg

## About

The Graceful boots (Hallowed) is a cosmetic recolor of the graceful boots, featuring a dark black and red color scheme. It is obtained by using a dark dye on regular graceful boots.

Dark dye can only be obtained from the Hallowed Sepulchre reward shop for 300 hallowed marks each. Six dyes are needed to recolor the complete graceful set.
