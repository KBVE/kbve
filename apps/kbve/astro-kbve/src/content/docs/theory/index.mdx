        - question: What is the Theory collection?
          answer: A set of essays and notes on the ideas behind KBVE — the craft of programming and automation, the practice of emulation and archiving dead code, and the cultural stances on social media, solarpunk, and open access that guide the work.
        - question: How is Theory different from the Application guides?
          answer: Application guides are hands-on field notes for specific tools. Theory is the why beneath them — principles, movements, and manifestos rather than commands and configs.
        - question: Can I contribute an essay?
          answer: Yes — every page is an MDX file in the open monorepo. Open a pull request against KBVE on GitHub and it ships with the next deploy.
        - question: Where do the tool and project docs live?
          answer: Tools live under /application/ and KBVE's own services and games under /project/, with the tagged hub at /projects/.
