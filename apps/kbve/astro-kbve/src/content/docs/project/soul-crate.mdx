## Overview

`Soul` is a Rust utility crate in the KBVE monorepo. It is maintained under
`packages/rust/soul` and published through the shared **crates** pipeline so it
can be reused across KBVE Rust projects and services. Versioning is tracked in
`version.toml` and released alongside the other workspace crates.
