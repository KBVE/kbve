    faq:
        - question: What is Kilobase?
          answer: Kilobase is KBVE's custom Supabase-based PostgreSQL image. It builds a pgrx Rust extension plus pg_failover_slots against PostgreSQL 17 and bakes them into a Supabase-compatible Postgres container.
        - question: What does the Kilobase pgrx extension add?
          answer: The kilobase extension exposes pgrx functions such as pgrx_extract_email and pgrx_extract_github_username that reuse jedi library zero-copy regex helpers as immutable, parallel-safe SQL functions.
        - question: What Postgres version does Kilobase target?
          answer: Kilobase builds against PostgreSQL 17 (pgdg dev-17) with the pg17 feature by default, using cargo-pgrx 0.16.1, and tags its image after the underlying supabase/postgres version.
