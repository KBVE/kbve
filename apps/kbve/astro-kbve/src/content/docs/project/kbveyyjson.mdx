jsonld:
    faq:
        - question: What is KBVEYYJson?
          answer: KBVEYYJson is an Unreal Engine 5 plugin that wraps yyjson 0.12.0, a high-performance C JSON library, so Unreal projects can parse and serialize JSON far faster than the engine's built-in JSON utilities.
        - question: Why use yyjson instead of Unreal's built-in JSON?
          answer: yyjson is one of the fastest C JSON libraries available. Wrapping it as a runtime module gives KBVE Unreal projects low-overhead JSON parsing and serialization, which matters for high-volume data such as ROWS backend payloads and codegen databases.
        - question: Which platforms and engine versions does KBVEYYJson support?
          answer: KBVEYYJson targets Unreal Engine 5.4 and later on Win64, Linux, and Mac. It is a third-party runtime module and is not enabled by default.
