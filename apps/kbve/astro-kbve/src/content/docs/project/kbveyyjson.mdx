## Overview

`KBVEYYJson` is an Unreal Engine 5 plugin that wraps
[yyjson](https://github.com/ibireme/yyjson) 0.12.0, a high-performance C JSON
library. It gives KBVE Unreal projects low-overhead JSON parsing and
serialization through a single runtime module, replacing the slower built-in
engine JSON utilities for high-volume payloads.

## Features

- **yyjson-backed** — wraps the yyjson 0.12.0 C library for fast JSON parsing and
  serialization.
- **Runtime module** — a single `KBVEYYJson` runtime module loaded in the default
  phase.
- **Cross-platform** — builds on Win64, Linux, and Mac under UE 5.4+.
- **Opt-in** — third-party category, not enabled by default, so projects pull it
  in only where fast JSON matters.

## FAQ

**What is KBVEYYJson?**
`KBVEYYJson` is an Unreal Engine 5 plugin that wraps yyjson 0.12.0, a
high-performance C JSON library, so Unreal projects can parse and serialize JSON
far faster than the engine's built-in JSON utilities.

**Why use yyjson instead of Unreal's built-in JSON?**
yyjson is one of the fastest C JSON libraries available. Wrapping it as a runtime
module gives KBVE Unreal projects low-overhead JSON parsing and serialization,
which matters for high-volume data such as ROWS backend payloads and codegen
databases.

**Which platforms and engine versions does KBVEYYJson support?**
KBVEYYJson targets Unreal Engine 5.4 and later on Win64, Linux, and Mac. It is a
third-party runtime module and is not enabled by default.
