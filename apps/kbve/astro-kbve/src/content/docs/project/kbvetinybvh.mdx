        - question: What is the KBVETinyBVH plugin?
          answer: KBVETinyBVH is an Unreal Engine wrapper for TinyBVH 1.6.7, a single-header BVH (bounding volume hierarchy) construction and traversal library used for ray tracing. It ships as a runtime third-party plugin.
        - question: What version of TinyBVH does it wrap?
          answer: The plugin wraps TinyBVH 1.6.7, matching the plugin VersionName, and bundles the tiny_bvh.h single-header library along with OpenCL and compute traversal kernels.
        - question: Which Unreal versions and platforms are supported?
          answer: KBVETinyBVH targets Unreal Engine 5.4 and newer and builds for Win64, Linux, and Mac. It is a runtime module and is not enabled by default.
