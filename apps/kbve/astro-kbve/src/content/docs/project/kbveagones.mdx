## Overview

Server-only glue plugin. Agones owns the GameServer lifecycle (Ready / Health);
`UKBVEAgonesBridgeSubsystem` listens for the Agones `ConnectedDelegate` and
registers the instance with ROWS via
`UROWSInstanceSubsystem::RegisterLauncher(Address, Port, MaxInstances)` using the
address/port Agones reports.

Depends on [Agones](./agones) and [KBVEROWS](./kbverows). The module is
`TargetAllowList: ["Server"]` and the subsystem gates on
`IsRunningDedicatedServer()`, so it stays out of client/editor builds. KBVEROWS
remains a pure REST plugin.
