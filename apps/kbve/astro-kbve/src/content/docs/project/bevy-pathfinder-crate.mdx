jsonld:
    faq:
        - question: What does the bevy_pathfinder crate do?
          answer: It provides grid-based flow-field pathfinding for tile worlds — a walkability grid, BFS-computed direction fields toward shared goals, and a narrow-passage (chokepoint) detector.
        - question: Why flow fields instead of per-agent A*?
          answer: When many agents share the same goal, a single BFS builds a direction field once and every agent looks up its next move in O(1), avoiding a separate A* search per agent.
        - question: Is Bevy required to use it?
          answer: No. The crate is pure Rust by default; enabling the optional bevy feature adds Resource derives for ECS integration.
