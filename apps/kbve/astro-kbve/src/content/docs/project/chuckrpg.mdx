## Overview

ChuckRPG is an open-world multiplayer RPG powered by Unreal Engine with OWS
(Open World Server) as its game backend. This project page tracks
`axum-chuckrpg` — the Rust [Axum](https://github.com/tokio-rs/axum) backend that
serves the ChuckRPG website, game API, and signed itch.io desktop downloads via
the shared `jedi` itch client.

## Features

- **Website + game API** — serves the ChuckRPG web frontend and backend API.
- **Signed itch.io downloads** — `/downloads/{platform}` redirects to the signed
  itch URL for the requested platform via the shared `jedi` itch client.
- **Downloads page** — `/downloads` renders an askama page of available builds.
- **Live build state API** — `/api/downloads` returns the current per-platform
  build state.
- **Health checks** — `/health` and `/api/health` for liveness probes.

## Usage

```
GET /downloads/{platform}   -> 302 redirect to signed itch.io URL
GET /downloads              -> HTML downloads page (askama)
GET /api/downloads          -> JSON per-platform build state
GET /health                 -> liveness
```

## FAQ

**What is ChuckRPG?**
ChuckRPG is an open-world multiplayer RPG powered by Unreal Engine with OWS
(Open World Server) as its game backend.

**What does the axum-chuckrpg service do?**
It is the Rust Axum API service that serves the ChuckRPG website, game API, and
resolves signed itch.io desktop downloads via the shared `jedi` itch client.

**How do desktop downloads work?**
A request to `/downloads/{platform}` redirects to the signed itch.io URL, while
`/api/downloads` exposes the live per-platform build state the web consumes.
