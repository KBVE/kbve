jsonld:
    faq:
        - question: What is KBVEROWS?
          answer: KBVEROWS is a unified Unreal Engine 5 backend integration plugin for the Rust Open World Server (ROWS). It provides auth, instance management, character persistence, and global data subsystems that talk to the ROWS REST API.
        - question: What subsystems does KBVEROWS provide?
          answer: The ROWS module ships subsystems including ROWSSubsystem, ROWSAuthSubsystem, ROWSCharacterSubsystem, and ROWSInstanceSubsystem, plus login UI widgets and a controller. Transport is pluggable via ROWSTransport with HTTP and gRPC implementations.
        - question: Which platforms and engine versions does KBVEROWS support?
          answer: KBVEROWS targets Unreal Engine 5.4 and later on Win64, Linux, and Mac. It is enabled by default and is currently a beta plugin.
