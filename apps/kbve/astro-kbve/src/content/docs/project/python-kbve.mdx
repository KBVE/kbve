    faq:
        - question: What is the kbve Python library?
          answer: kbve is an API layer and complexity handler for async HTTP, WebSocket, gRPC, and broadcasting. It provides the core modules that packages like fudster build on, spanning api, grpc, proto, server, and health.
        - question: How do I install the kbve Python package?
          answer: Install it from PyPI with pip install kbve. It requires Python 3.12 or newer and depends on pydantic, aiohttp, fastapi, uvicorn, grpcio, and protobuf.
        - question: What optional extras does kbve provide?
          answer: kbve ships opt-in extras kept off the core install — sprite for the ARPG sprite-baking toolchain (Pillow, numpy), osrs for OSRS item-corpus analysis, and seo for the whole-site SEO/SEM auditor — each enabled with uv sync --extra.
