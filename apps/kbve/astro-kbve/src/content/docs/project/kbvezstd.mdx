        - question: What is the KBVEZstd plugin?
          answer: KBVEZstd is an Unreal Engine runtime plugin that wraps Facebook's Zstandard (zstd) 1.5.7 compression library, exposing fast real-time lossless compression to Unreal projects on Win64, Linux, and Mac.
        - question: Which Unreal Engine versions does KBVEZstd support?
          answer: KBVEZstd targets a minimum engine version of 5.4 and ships for the Win64, Linux, and Mac platforms. It is a Runtime module loaded at the Default loading phase.
        - question: Why use Zstandard in an Unreal project?
          answer: Zstandard offers a strong compression-ratio to speed trade-off suited to real-time workloads, making it well matched to compressing save data, world chunks, and network payloads in a game.
