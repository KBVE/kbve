## Overview

Game-agnostic world/map data + Mass bridge. `UKBVEMapDatabase` (GameInstance
subsystem) loads `Content/Data/mapdb-data.json` into `FKBVEWorldObjectDef` records
(resource nodes, containers, stations — harvest yield/amount/time, spawn weights),
ref lookup. `FKBVEWorldObjectFragment` + `FKBVEWorldObjectTag` carry a placed
object's identity + harvest state into Mass for harvestable world entities.
