jsonld:
    faq:
        - question: What is the KBVEMapDB plugin?
          answer: KBVEMapDB is a game-agnostic Unreal Engine plugin providing world/map data plus a Mass bridge. The UKBVEMapDatabase GameInstance subsystem loads the shared mapdb-data.json artifact into FKBVEWorldObjectDef records and exposes ref lookup.
        - question: What world-object data does KBVEMapDB load?
          answer: It loads FKBVEWorldObjectDef records for resource nodes, containers, and stations, including harvest yield, amount, and time plus spawn weights, from Content/Data/mapdb-data.json.
        - question: How does KBVEMapDB integrate with Mass?
          answer: FKBVEWorldObjectFragment and FKBVEWorldObjectTag carry a placed object's identity and harvest state into Mass, so harvestable world entities are driven by the same shared definitions.
