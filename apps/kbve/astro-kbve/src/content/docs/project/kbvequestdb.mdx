## Overview

Game-agnostic quest catalog. `UKBVEQuestDatabase` (GameInstance subsystem) loads
`Content/Data/questdb-data.json` into `FKBVEQuestDef` records (objectives, rewards,
category, chain), ref lookup. Quest progress/state is game-side. Mirrors the
itemdb/npcdb loader pattern.
