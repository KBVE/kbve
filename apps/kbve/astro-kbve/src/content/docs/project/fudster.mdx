    faq:
        - question: What is Fudster?
          answer: Fudster is a composable Python ML library and CLI built around the kbve core, focused on haystack-ai and pgvector integrations. It ships API models, async HTTP/WebSocket clients, and a fudster command-line entry point.
        - question: How do I install Fudster?
          answer: Install it from PyPI with pip install fudster. It requires Python 3.12 or newer and pulls in kbve, pydantic, fastapi, aiohttp, and related async and parsing dependencies.
        - question: What optional extras does Fudster provide?
          answer: Fudster offers opt-in extras kept off the core install — automation (opencv, pyautogui, humancursor), browser (seleniumbase), ocr (pytesseract), and vnc (websockify, websockets) — plus an all extra that bundles them together.
