## Overview

KBVEWASM wraps the WebAssembly Micro Runtime (WAMR) 2.2.0 for Unreal Engine, letting UE projects load and execute sandboxed WebAssembly modules from inside the engine. It embeds WAMR — a lightweight, portable WebAssembly runtime — so game and tooling logic authored in any Wasm-targeting language can run within UE across Win64, Linux, and Mac.

## Features

- **WAMR-backed runtime** — embeds WebAssembly Micro Runtime 2.2.0 for executing Wasm modules in-engine.
- **Sandboxed execution** — runs WebAssembly modules in an isolated runtime rather than native UE code.
- **Cross-platform** — supported on Win64, Linux, and Mac under Unreal Engine 5.4+.

## FAQ

**What does the KBVEWASM plugin do?**
KBVEWASM wraps the WebAssembly Micro Runtime (WAMR) 2.2.0 for Unreal Engine, letting UE projects load and run sandboxed WebAssembly modules from inside the engine.

**Which runtime does KBVEWASM use?**
It uses WAMR (WebAssembly Micro Runtime) version 2.2.0, a lightweight embeddable WebAssembly runtime, matching the plugin version.

**Which platforms and engine versions does KBVEWASM support?**
KBVEWASM targets Unreal Engine 5.4 and later on Win64, Linux, and Mac.
