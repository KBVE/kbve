## Overview

Pristine mirror of the upstream Agones Unreal SDK (googleforgames/agones,
`release-1.58.0`, Apache-2.0), pinned for the KBVE dedicated-server stack. The
`UAgonesSubsystem` auto-connects on dedicated-server builds — polls
`/gameserver`, calls `/ready`, and pings `/health` — so a server binary becomes
allocatable under an Agones Fleet.

Server-only: the module is `TargetAllowList: ["Server"]`, so it is never
compiled into client/editor builds. The KBVE-specific ROWS wiring lives in the
separate [KBVEAgones](./kbveagones) plugin. See `VENDORING.md` in the plugin for
the upstream pin and resync steps.
