jsonld:
    faq:
        - question: What is the Agones Unreal SDK plugin?
          answer: It is a pristine vendored mirror of the upstream googleforgames/agones Unreal SDK (release-1.58.0, Apache-2.0), pinned for the KBVE dedicated-server stack. It wraps the Agones sidecar REST API so an Unreal dedicated server can register itself with an Agones Fleet.
        - question: How does the plugin manage the GameServer lifecycle?
          answer: The UAgonesSubsystem auto-connects on dedicated-server builds. It polls /gameserver, calls /ready, and pings /health against the Agones sidecar so the server binary becomes allocatable and stays healthy under a Fleet.
        - question: Is the Agones plugin compiled into client or editor builds?
          answer: No. The module declares TargetAllowList ["Server"], so it is only built into dedicated-server targets and never into client or editor builds.
