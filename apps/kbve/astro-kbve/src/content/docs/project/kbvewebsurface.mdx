        - question: What is the KBVEWebSurface plugin?
          answer: KBVEWebSurface is a game-agnostic Unreal Engine plugin that renders interactive web content onto world surfaces — flat panels, curved or holographic meshes, and drop-in terminal or kiosk actors — for reuse across chuckrpg, future KBVE titles, and external Fab distribution.

          answer: Avoid it for combat HUDs, health bars, and minimaps — use Slate or UMG instead — and for per-NPC labels, shop signs, or hundreds of world objects, where a CEF surface per object is too expensive.

    features:
        - title: Flat panel
          icon: render
          body: UKBVEWebSurfaceComponent wraps a UWidgetComponent with an embedded UMG WebBrowser.
        - title: Curved / holographic
          icon: expand
          body: UKBVEWebRenderSurfaceComponent pipes the browser through a render target onto an arbitrary mesh + material.
        - title: Actor
          icon: box
          body: AKBVEWebSurfaceActor ships as a drop-in for terminals and kiosks.
        - title: Input
          icon: sensor
          body: UKBVEWebInputRouter maps player line traces to widget-pixel coordinates. Hit UVs come from FindCollisionUV, so Support UV From Hit Results must be enabled on the project.
        - title: Bridge
          icon: network
          body: UKBVEWebBridge exposes a typed message bus between embedded JS and Blueprint/C++. Pages call window.kbveBridge.send(channel, payload); the bridge re-broadcasts via OnMessage.
        - title: Performance
          icon: zap
          body: Per-surface frame-rate cap, offscreen pause, snapshot fallback at distance, and a UKBVEWebSurfacePool LRU cap on concurrent live surfaces.
        - title: Backend
          icon: settings
          body: The default backend wraps Unreal's stock UWebBrowser (CEF). Alternate backends (Ultralight, WebUI) plug in via IKBVEWebBackend as separate plugins.

### Use cases

- Marketplace / auction house
- Account, guild, social screens
- Admin and debug terminals
- Patch notes, news boards
- In-world interactive billboards and kiosks

### Anti-use cases

- Combat HUD, health bars, minimap — use Slate/UMG.
- Per-NPC labels, shop signs, hundreds of world objects — too expensive per surface.
