jsonld:
    faq:
        - question: What does the bevy_statemachine crate do?
          answer: It is a thread-safe state snapshot bridge for Bevy that exposes ECS resources to external consumers such as Tauri IPC, WASM JavaScript, and lightyear networking, with both JSON and bincode serialization.
        - question: How do you read a state snapshot from bevy_statemachine?
          answer: Add StateSnapshotPlugin generic over your state type, then call get_snapshot to retrieve the latest snapshot, get_snapshot_json for a JSON string, or snapshot_version for the monotonic version counter.
        - question: What serialization formats does bevy_statemachine support?
          answer: The default serde feature provides JSON serialization via serde_json, and the optional bincode feature adds binary serialization for compact transport.
