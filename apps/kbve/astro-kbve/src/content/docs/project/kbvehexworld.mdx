    faq:
        - question: What is KBVEHexWorld?
          answer: KBVEHexWorld is the hex-grid world topology module of the KBVEWorld Unreal plugin. It provides deterministic world seeds, noise-driven terrain, and a foundation for PCG streaming, exposing a Blueprint-callable master map of hex cells.
        - question: Is KBVEHexWorld a standalone plugin?
          answer: No. It is now a module inside the KBVEWorld plugin rather than a standalone plugin. Enable KBVEWorld to get both the square-chunk world module and the hex topology module together.
        - question: How do you query the hex world?
          answer: The UHexWorldSubsystem is a GameInstanceSubsystem that maintains a master hex map. It exposes Blueprint-callable functions to register and unregister cells by FHexCoord, look up a cell record, list all coordinates, and query a cell's neighbors.
