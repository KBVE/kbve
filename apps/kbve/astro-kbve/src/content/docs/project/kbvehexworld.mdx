## Overview

`KBVEHexWorld` is the hex-grid world topology module of the **KBVEWorld** Unreal
plugin. It provides deterministic world seeds, noise-driven terrain, and a
foundation for PCG streaming. The `UHexWorldSubsystem` maintains a master map of
hex cells with Blueprint-callable registration and query functions.

## Features

- **Hex topology** — hex-grid coordinates (`FHexCoord`) and cell records
  (`FHexCellRecord`) with neighbor queries.
- **Deterministic seeds** — `HexWorldSeedSubsystem` for reproducible world
  generation.
- **Noise terrain** — `HexNoiseSubsystem` and `HexEnvironmentSubsystem` drive
  procedural terrain and environment.
- **Blueprint-callable map** — `UHexWorldSubsystem` (a `GameInstanceSubsystem`)
  registers, unregisters, looks up, and enumerates hex cells.

## FAQ

**What is KBVEHexWorld?**
`KBVEHexWorld` is the hex-grid world topology module of the KBVEWorld Unreal
plugin. It provides deterministic world seeds, noise-driven terrain, and a
foundation for PCG streaming, exposing a Blueprint-callable master map of hex
cells.

**Is KBVEHexWorld a standalone plugin?**
No. It is now a module inside the KBVEWorld plugin rather than a standalone
plugin. Enable `KBVEWorld` to get both the square-chunk world module and the hex
topology module together.

**How do you query the hex world?**
The `UHexWorldSubsystem` is a `GameInstanceSubsystem` that maintains a master hex
map. It exposes Blueprint-callable functions to register and unregister cells by
`FHexCoord`, look up a cell record, list all coordinates, and query a cell's
neighbors.
