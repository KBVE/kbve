jsonld:
    faq:
        - question: What does the bevy_cam crate do?
          answer: It provides an isometric camera plugin for Bevy 0.18 that renders the 3D scene to a low-resolution texture at a fixed pixel density, then upscales it with nearest-neighbor sampling for crisp pixel-art output.
        - question: How does the camera follow a target?
          answer: Attach the CameraFollowTarget component to any entity. The scene camera tracks it and snaps to the pixel grid on all three camera-space axes to eliminate sub-pixel jitter.
        - question: Is scroll-wheel zoom supported?
          answer: Yes, behind the optional "zoom" feature flag. Zoom is multiplicative — each notch scales by a fixed percentage and interpolates smoothly toward the target — and is off by default to avoid wasted system dispatch on fixed-camera games.
