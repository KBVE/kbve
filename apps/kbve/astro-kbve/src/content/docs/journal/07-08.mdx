### AstroPad Supabase Services

-   04:55PM

    The goal for today will be to build out all the service files for the supabase auth system.
    This singleton approach should help us with a bit better memory management and make things cleaner to use.
    Our first three services will be the login, register and the oauth, which will be the 5th time we done a rewrite of it within the past three years.    

    The three services will be `ServiceLogin`, `ServiceRegister` and `ServiceOAuth` and majority of the code for these have been written out.
    We are just cleaning this up and applying some level of optimization!
    The end goal will be that we can have multiple sites that can quickly integrate into our eco-system, making future websites easier to manage and scale up.
    I am excited to see how it goes about building this out, being able to mass produce websites is going to be a great feature for us.
