### OSRS Over Allocation Issue

-   04:45PM

    Updating the OSRS axum routes, mainly reducing the allocation from 4 strings down to a single one.
    However we can not go zero allocation because of the mpsc, which might be an issue in the future.
    There is a way we could avoid the cache actor and directly hook into the dashmap, but that might hurt our scaling.
    For reference, we are using the tokio's sync mpsc rather than the std mpsc channel. 

### Content Margin/Padding

-   07:29PM

    Finally figured out the issue with the starlight theme causing some weird element issues!
    Turns out that I need to add the `not-content` css tag to anything that might conflict with our starlight theme, allowing us to just not have any of their overrides.
    This should solve some of the button issues we had earlier as well.
    Going to implement this fix, hopefully it will resolve the other margin issues with the OSRS.
