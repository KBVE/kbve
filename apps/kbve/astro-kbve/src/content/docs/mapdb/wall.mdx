# Wall

A single-hex stone rampart. Crenellated along the top, arrow slits flanking a central arched gate, and dark bastion stubs at each end so the segment terminates cleanly when it meets a Tower or a Keep. Low enough that units still read above it in the silhouette; tall enough that the threat is obvious.

## Game Role

- **Standalone build** — no upgrade chain, place directly inside Player territory for 5 Stone Block.
- **Barrier + gate** — tagged with `WallTag` for the future pathing-block pipeline; shader shows the gate arch for lore continuity even before the traversal logic lands.
- **Chainable** — designed to bridge Towers into a perimeter.
- **Footprint** — single hex.
- **Durability** — 260 HP.

## Lore

The first Wall in a kingdom is never the one that stops an army. It is a declaration. Whoever commissioned it decided, for the first time, that there was a difference between *inside* and *outside* worth marking in stone. Every Wall that follows argues the same line a little louder.
