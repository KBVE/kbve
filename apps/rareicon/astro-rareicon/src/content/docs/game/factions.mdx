RareIcon's world has competing factions with their own building palettes, production chains, and combat identity. Early iterations focus on a single playable civilization so the core simulation stays tight; additional factions roll in as systems stabilize.

For the current work-in-progress roster, see the [dev journal](https://kbve.com/journal/) on kbve.com.
