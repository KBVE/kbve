## Pitch

> Chip back at the system. RareIcon is a 2D sci-fi action-RPG bullet-hell
> roguelite set in a collapsing digital dystopia. Play as Chip, a rogue
> process fighting through DaemonCorps territory using stolen icons,
> recursive abilities, and a kingdom of allied units that build, gather,
> and fight beside you between runs.

## Description

RareIcon mixes the fast-paced bullet-hell of arcade shoot-'em-ups with
the long-tail systems of a kingdom-building RPG. Each run sends Chip
into a procedurally-corrupted shard of the network — clear waves,
recover icons, and bring the loot home to upgrade your base. Recruit
specialists, assign them roles (builder, gatherer, fighter, scout),
and watch the kingdom take care of itself between sorties.

The DaemonCorps wants the stack back. They will keep coming.

## Key Features

- **Bullet-hell first.** Tight 2D combat. Hundreds of enemies, no
  framerate excuses — built on Unity DOTS/ECS for a stable 60 FPS
  even at high entity counts.
- **Run + return.** Each sortie funnels resources back into a
  persistent kingdom. Death isn't a reset — it's a checkpoint.
- **Multi-role units.** Every unit can builder / gather / fight / scout;
  specialization is a priority weight, not a class lock-in.
- **Hex-grid world.** Sheltered livestock, hive bees, and pond fish
  collapse into host-tile counts so the world keeps scaling.
- **Recursive abilities.** Stolen icons stack. The build that broke
  the last run might break the next one harder.
- **Open-source ecosystem.** Built in the open at
  [github.com/kbve/kbve](https://github.com/kbve/kbve).
