## Install

1. Open Steam and search for **RareIcon** (store link will go here at launch).
2. Click **Download Demo**.
3. Launch from your Steam library once the install finishes.

## First run

On first launch the game will create a save folder under your OS user directory and generate a starting map. If the game fails to start, verify local files through Steam and check the [Controls](/guides/controls/) page for input setup.
