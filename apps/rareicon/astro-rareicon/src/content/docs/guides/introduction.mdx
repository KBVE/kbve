**RareIcon** is a hex-grid colony simulation built in Unity with the DOTS / ECS stack. You start with a handful of settlers on an empty patch of world and grow it — farms, villages, barracks, keeps, castles — while handling the things that go wrong in between.

## Pillars

- **Colony-sim depth.** Every unit has job priorities. Every building is a bundle of capability components. Production chains, tenders, logistics, combat — all scheduled on the same tick loop.
- **DOTS performance.** Target scale is thousands of units across thousands of tiles. Burst + parallel jobs drive the core systems; no per-entity managed overhead in the hot path.
- **Readable state.** The UI surfaces what a settler is doing and why: which task they picked, what the dispatcher scored, how much of a resource a bank holds.
