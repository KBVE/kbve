RareIcon uses **WASD + mouse** as its core twin-stick scheme, layered with spells and auto-aim support so you can adapt mid-chaos.

## Movement

| Action | Key |
| --- | --- |
| Move | `W` `A` `S` `D` |
| Dash / evade | `Shift` |
| Interact | `E` |
| Reload / swap magazine | `R` |

## Combat

| Action | Key |
| --- | --- |
| Aim | Mouse |
| Fire | Left mouse |
| Alt-fire / focus | Right mouse |
| Toggle auto-aim | `T` |
| Cast spell (slot 1-4) | `1` `2` `3` `4` |
| Cycle spells | `Q` |

## Party

| Action | Key |
| --- | --- |
| Command companion | `C` |
| Party panel | `P` |

## UI

| Action | Key |
| --- | --- |
| Inventory | `Tab` |
| Map | `M` |
| Pause | `Esc` |
| Sanctuary / base menu | `B` |
