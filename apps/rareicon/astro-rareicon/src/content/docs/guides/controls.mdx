## Camera

| Action | Key |
| --- | --- |
| Pan | `W` `A` `S` `D` / middle-mouse drag |
| Zoom | Scroll wheel |
| Rotate | `Q` / `E` |
| Recenter on selection | `F` |

## Selection

| Action | Key |
| --- | --- |
| Select unit / building | Left click |
| Box select | Left click + drag |
| Add to selection | `Shift` + click |
| Clear selection | `Esc` |

## Time

| Action | Key |
| --- | --- |
| Pause / resume | `Space` |
| 1× / 2× / 3× speed | `1` / `2` / `3` |

## UI

| Action | Key |
| --- | --- |
| Build menu | `B` |
| Job priorities | `J` |
| Overview panel | `Tab` |
