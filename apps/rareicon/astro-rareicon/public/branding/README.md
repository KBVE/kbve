# RareIcon Brand Assets

Drop the following files in this directory to populate the public press kit:

| File                        | Format | Size              | Use                                |
| --------------------------- | ------ | ----------------- | ---------------------------------- |
| `logo-wordmark.svg`         | SVG    | viewBox 0 0 ?? ?? | Vector wordmark, recolor via fill  |
| `logo-glyph.svg`            | SVG    | viewBox 0 0 24 24 | Standalone glyph, recolor via fill |
| `logo-wordmark-light.png`   | PNG    | 2x — 1024 wide    | Wordmark on dark backgrounds       |
| `logo-wordmark-dark.png`    | PNG    | 2x — 1024 wide    | Wordmark on light backgrounds      |
| `key-art-3840x2160.png`     | PNG    | 4K (16:9)         | Press hero, store header backdrop  |
| `key-art-1920x1080.png`     | PNG    | 1080p (16:9)      | Article hero / blog post header    |
| `screenshot-01.png` … 10    | PNG    | 1920×1080         | Gameplay screenshots               |
| `og-card-1200x630.png`      | PNG    | 1200×630          | Open Graph default share card      |
| `twitter-card-1200x675.png` | PNG    | 1200×675          | Twitter / X large summary card     |
| `palette.svg`               | SVG    | —                 | Brand color swatches               |

## Permitted use

Press, streamers, and content creators are cleared to use any asset in
this directory in coverage of RareIcon (reviews, articles, video
thumbnails, stream overlays). Attribution appreciated but not required.
Do not modify the wordmark proportions or recolor inside the icon
container — apply color via the `currentColor` token on SVG variants.

For higher-resolution assets, raw footage, or capsule renders, contact
`press@kbve.com`.
