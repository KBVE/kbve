## Getting Started

Welcome to KBVE MC. This guide will help you connect and get started.

## Connecting to the Server

1. Open Minecraft (Java Edition)
2. Click **Multiplayer** > **Add Server**
3. Enter the server address: `mc.kbve.com`
4. Click **Done** and join!

## Server Rules

- Be respectful to other players
- No griefing or stealing
- No exploits or cheating
- Have fun and build something amazing
